Introduction

HTML (HyperText Markup Language) is the standard language for creating webpages. It structures your content and elements like text, images, and links. It forms the backbone of all websites on the internet before the application of CSS.

Introduction to Text Editors, and Web Browsers

Text Editors: Writing HTML Code

To create HTML files, you need a text editor—a software application that allows you to write and edit text. Popular text editors include:

  • Notepad (Windows): A basic text editor that comes pre-installed with Windows.
  • TextEdit (Mac): A simple text editor available on macOS.
  • VS Code: A popular, free code editor that provides advanced features for coding in HTML and other languages.
  • Sublime Text: A lightweight text editor that supports a wide range of programming languages.
  • Atom: An open-source text editor with customizable features.

Simply open any of these text editors and start writing your HTML code. Once you're done, you can save the file with a .html extension (e.g., index.html).

Web Browsers: Viewing HTML Files

Web browsers are software applications used to view and interact with HTML files. Browsers interpret the HTML code and render it as a visual webpage. Popular web browsers include:

  • Google Chrome: A fast and widely-used browser known for its developer tools.
  • Mozilla Firefox: A browser focused on privacy and open-source development.
  • Microsoft Edge: A browser developed by Microsoft, based on Chromium.
  • Safari: The default web browser for Apple devices, known for its speed and efficiency.

To view an HTML file, simply open it in any of these web browsers. The browser will process the HTML code and display it as a formatted webpage.

How HTML Works

HTML works by using a series of elements, which are defined by tags. These tags are enclosed in angle brackets <> and come in pairs: an opening tag <tag> and a closing tag </tag>. The content placed between these tags is affected by the tags' properties.

Here’s a basic example of HTML code:

<!DOCTYPE html>
<html>
    <head>
        <title>My First Webpage</title>
    </head>
    <body>
        <h1>Welcome to My Webpage</h1>
        <p>This is a paragraph of text on my webpage.</p>
    </body>
</html>

The above code includes the following elements:

  • <!DOCTYPE html>: Declares the document type and version of HTML being used.
  • <html>: The root element that wraps all the content on the page.
  • <head>: Contains meta-information about the document, such as its title.
  • <title>: Sets the title of the webpage, which appears in the browser tab.
  • <body>: Contains the content that will be visible on the webpage, including headings and paragraphs.
  • <h1>: Represents a top-level heading.
  • <p>: Defines a paragraph of text.

When this code is opened in a web browser, it will display a webpage with a title and a heading followed by a paragraph of text.

Understanding HTML Tags

In HTML, tags are used to define the structure and content of your web page. An HTML tag is usually enclosed in angle brackets, <>, and comes in pairs: an opening tag <tagname> and a closing tag </tagname>.

The content between the opening and closing tags is affected by the tag's properties. For example, consider the following HTML snippet:

<p>This is a paragraph.</p>

Here, <p> is the opening tag for a paragraph, and </p> is the closing tag. The content "This is a paragraph." is enclosed within these tags and will be displayed as a paragraph on the webpage.

Tags often have attributes that provide additional information or modify the behavior of an element. For instance:

<a href="https://www.example.com">Click here</a>

In this example, the <a> tag is used to create a hyperlink, with the href attribute specifying the URL.

Understanding Common HTML Elements

Headings: <h1> to <h6>

HTML provides six levels of headings, from <h1> (the most important) to <h6> (the least important). Headings are typically used for titles and subtitles.

<h1>This is a level 1 heading</h1>
<h2>This is a level 2 heading</h2>
<h3>This is a level 3 heading</h3>
<h4>This is a level 4 heading</h4>
<h5>This is a level 5 heading</h5>
<h6>This is a level 6 heading</h6>

Centering Text: <center>

The <center> tag is used to center-align text and other content. However, it is important to note that this tag is deprecated in HTML5. Instead, use CSS for centering content.

<center>This text is centered</center>

Underlining Text: <u>

The <u> tag is used to underline text.

<u>This text is underlined</u>

Italicizing Text: <i>

The <i> tag is used to italicize text.

<i>This text is italicized</i>

Preformatted Text: <pre>

The <pre> tag is used to display preformatted text. This means that whitespace and line breaks are preserved as they are in the HTML code.

<pre>
This is preformatted text.
    Line breaks and spaces
    are preserved in this block.
</pre>

Paragraphs: <p>

The <p> tag defines a paragraph of text. Paragraphs are block-level elements and are typically used to group sentences together.

<p>This is a paragraph of text.</p>

Understanding HTML Lists: <ul> and <ol>

Unordered Lists: <ul>

An unordered list is a collection of items that do not have a specific order. Each item in the list is usually marked with a bullet point.

Here is an example of a basic unordered list:

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

The above code will render as:

  • Item 1
  • Item 2
  • Item 3

Ordered Lists: <ol>

An ordered list is a collection of items that have a specific sequence. Each item is numbered by default.

Here is an example of a basic ordered list:

<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>

The above code will render as:

  1. First item
  2. Second item
  3. Third item

Ordered Lists with Custom Types

You can customize the numbering style of an ordered list using the type attribute. For example:

Alphabetical list: Using type="a" will create a list with lowercase alphabetical characters.

<ol type="a">
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>

The above code will render as:

  1. First item
  2. Second item
  3. Third item

Roman numeral list: Using type="i" will create a list with lowercase Roman numerals.

<ol type="i">
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>

The above code will render as:

  1. First item
  2. Second item
  3. Third item

Unordered Lists with Custom Bullet Types

While you cannot directly change the bullet style using the type attribute with <ul>, you can control the appearance using CSS. However, for demonstration, let's consider the default bullet style:

Default bullet list:

<ul>

    <li>Third item</li>
</ul>

Understanding HTML Tables: <table>, <tr>, and <td>

What is a Table in HTML?

An HTML table is used to organize data into rows and columns. It is a powerful tool for displaying tabular data in a structured format.

Table Rows: <tr>

The <tr> tag defines a row in an HTML table. A table is composed of one or more rows, each represented by a <tr> tag.

For example, this code defines a table with two rows:

<table>
    <tr>
        <td>Row 1, Cell 1</td>
        <td>Row 1, Cell 2</td>
    </tr>
    <tr>
        <td>Row 2, Cell 1</td>
        <td>Row 2, Cell 2</td>
    </tr>
</table>

Table Data: <td>

The <td> tag defines a cell within a table row. Cells are the individual boxes that contain data within a table.

For example, within a row, each <td> tag represents a separate cell:

<tr>
    <td>This is a cell</td>
    <td>This is another cell</td>
</tr>

Putting It All Together: Creating a Simple Table

Here’s an example of how you can create a simple HTML table using the <table>, <tr>, and <td> tags:

<table border="1">
    <tr>
        <td>Name</td>
        <td>Age</td>
    </tr>
    <tr>
        <td>Alice</td>
        <td>30</td>
    </tr>
    <tr>
        <td>Bob</td>
        <td>25</td>
    </tr>
</table>

The above code will render as:

Name Age
Alice 30
Bob 25

Summary

In summary, the <tr> tag defines a row in the table, and the <td> tag defines each cell within that row. Together, these elements allow you to create structured tables in HTML.

Adding Images, Videos, and Audio in HTML

Working with Images: <img> Tag

The <img> tag is used to embed images in an HTML document. This tag is self-closing, meaning it doesn’t require a closing tag.

The basic syntax for adding an image looks like this:

<img src="image.jpg" alt="Description of the image">

Here’s an example of how to add an image:

<img src="logo.jpg" alt="Marhoboss logo">

Why Keep Media Files in the Same Directory as Your HTML Document?

When you reference an image, video, or audio file in your HTML code using the src attribute, the browser needs to know where to find that file. If the media files are stored in the same directory (folder) as your HTML document, you can easily link them using just the file name (e.g., cat.jpg).

If the files are in a different directory, you need to provide the correct relative path. However, keeping media files in the same directory simplifies the process and reduces the chances of errors when linking to these files.

For example:

  • If your HTML file is located in a folder called website, and your image is also in that same folder, you can link it like this:
    • <img src="Orhosere.jpg" alt=" learn">
  • If your image is in a subfolder called images within the website folder, you need to specify the path like this:
    • <img src="images/Orhosere.jpg" alt="learn ">

Keeping your HTML file and media files in the same directory ensures that your links are simple and straightforward, which is especially useful when moving your project to different locations or sharing it with others.
